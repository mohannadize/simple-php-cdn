API Documentation

Upload Image

POST /upload
Content-Type: multipart/form-data
Form field: image

Access Image

GET /image/{filename}?w=500&q=80

Parameters:

  • w: Desired width in pixels (optional)
  • q: JPEG quality 1-100 (optional)
